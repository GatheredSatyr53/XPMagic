# XPMagic

Мод для Minecraft, позволяющий сохранять свой опыт в бутылки и выпивать его обратно.

[Начат](https://github.com/Avvros/XPMagic-Legacy) для Minecraft 1.12.2, портирован на Minecraft 26.2 (Forge).

## Содержимое мода

- **Хранитель опыта** (XP Keeping Machine) — печкоподобная машина. Кладёте стеклянную бутылочку,
  топливо и порошок памяти, держите интерфейс открытым — машина забирает у вас 10 очков опыта
  и наполняет бутылочку. Работает только пока меню открыто; без вас встаёт на паузу.
- **Коктейль опыта** (XP Cocktail) — результат работы машины. Хранит опыт в компоненте данных,
  показывает количество в подсказке и возвращает опыт при выпивании.
- **Порошок памяти** и **Чип обработки** — промежуточные предметы для крафта.

Локализации: английская и русская.

## Структура репозитория

| Папка | Что это |
|---|---|
| `26.2/` | Актуальная версия мода: Minecraft 26.2, Forge 65+, Java 25 |

## Сборка и запуск

Из папки `26.2`:

```
gradlew build      # собрать jar в build/libs
gradlew runClient  # запустить клиент с модом
gradlew runData    # перегенерировать данные (рецепты, loot tables, теги)
```

Рецепты, loot tables и теги генерируются кодом (`src/main/java/.../datagen/`) в
`src/generated/resources` — после изменения провайдеров запустите `runData` и закоммитьте
результат. Модели, текстуры и локализации лежат руками в `src/main/resources`.
